# 🍽️ Mess Management System

A full-stack web app for managing mess operations.

## 🚀 Features
- Email + OTP Authentication
- Google Login
- Student Dashboard
- Menu, Feedback, Complaints
- JWT Authentication

## 🛠️ Tech Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database: MongoDB Atlas
- Auth: Firebase + JWT + OTP
